## ⚠️ 常见错误和解决方案

### 错误 1：显示 warp=on 而不是 warp=plus

```
问题：
┌─────────────────────────┐
│ $ warp-cli trace        │
│ warp=on  ✗              │
└─────────────────────────┘

原因：
账户在绑定 key 之前已经连接过 VPN

解决方案：
┌─────────────────────────┐
│ 1. 注册新账户            │
│ $ warp-cli register     │
│                         │
│ 2. 立即绑定 key         │
│ $ warp-cli update \     │
│   --license-key "..."   │
│                         │
│ 3. 然后连接             │
│ $ warp-cli connect      │
└─────────────────────────┘
```

### 错误 2：设备数量超限

```
问题：
┌─────────────────────────┐
│ Error: Maximum devices  │
│ reached (5/5)           │
└─────────────────────────┘

解决方案：
┌─────────────────────────┐
│ 1. 查看设备列表          │
│ $ warp-cli status       │
│                         │
│ 2. 移除旧设备           │
│ $ warp-cli update \     │
│   --remove "device-id"  │
└─────────────────────────┘
```

## 🎯 快速命令参考

```bash
# 查看完整帮助
warp-cli update --help

# 绑定 license key
warp-cli update --license-key "YOUR_KEY"

# 设置设备名称
warp-cli update --name "My-Laptop"

# 查看账户状态
warp-cli status

# 验证 Warp+ 状态
warp-cli trace

# 移除设备
warp-cli update --remove "device-id"
```
